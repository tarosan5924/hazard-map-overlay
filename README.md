# Hazard Map Overlay for Real Estate

不動産ポータルサイトの物件詳細ページに、ハザードマップを自動で重畳表示する Chrome 拡張機能（Manifest V3）。

物件探索の初期段階で「広域が危険区域である物件」を視覚的に除外する粗いフィルターとして使う。

## 対応サイト

- [SUUMO](https://suumo.jp/)（物件詳細ページ）

## 表示されるハザード情報

- 洪水浸水想定区域（想定最大規模）
- 土砂災害警戒区域

データソース: [国土交通省ハザードマップポータルサイト](https://disaportal.gsi.go.jp/)

## インストール手順

1. このリポジトリを clone する

   ```bash
   git clone https://github.com/tarosan5924/hazard-map-overlay.git
   ```

2. Chrome で `chrome://extensions/` を開き、右上の「デベロッパーモード」を有効にする

3. 「パッケージ化されていない拡張機能を読み込む」をクリックし、`hazard-map-extension/` ディレクトリを選択する

4. SUUMO の物件詳細ページを開くと、住所欄の下にハザードマップが表示される

## セレクタのカスタマイズ

拡張機能アイコンを右クリック →「オプション」から、サイトごとの住所抽出セレクタと挿入先セレクタを変更できる。

JSON エクスポート / インポートでバックアップや共有が可能。

## 開発

ビルドステップ不要（バニラ JS）。

```bash
# 依存パッケージインストール
pnpm install

# テスト
pnpm test

# lint / フォーマット
pnpm biome check src/
```

## 既知の制限事項

- **座標精度**: 国土地理院のジオコーディング API が返す座標はエリア代表点。物件の正確な位置を示すものではない。
- **ジオコーディング API**: 無償のベストエフォート提供（SLA なし）。住所によっては座標を取得できない場合がある。
- **AJAX 遅延**: 物件情報が JavaScript で動的に描画されるページでは、表示まで数秒かかることがある。
